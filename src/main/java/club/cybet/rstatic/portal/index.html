Turing's HCPCS Level II Natural Language Search Engine (excluding D-codes), with rich support for the Lucene Query Parser syntax, is an application for searching HCPCS procedure codes and descriptions for a specified word or phrase. A blank search returns all results.

Turing's Medicare Part A Skilled Nursing Facility Pricer is a calculator for estimating per diem payments under the Prospective Payment System. It supports all dates of service from July 1st, 1998 thru today.

Turing's Fee Schedule Look-Up Tool enables searching for pricing amounts associated with a HCPCS Level II procedure code (excluding D-codes) for Medicare Part B.

Turing's implementation of the National Correct Coding Initiative's Medically Unlikely Edits for Durable Medical Equipment HCPCS procedure codes is a lookup tool for determining the maximum units of service typically allowed per diem for a single beneficiary.
